# 📚 LuminaReads - Book Reviews & Literary Insights

**LuminaReads** is a simple, static web application designed as a my university project 

🔗 **Live Demo:**  https://sachintha123-scl.github.io/LuminaReads-/

---

## ✨ Features

- **Trending & Recent Reviews:** Discover the most celebrated and highly-rated book reviews of the month.
- **Browse Catalog:** Easily navigate through a growing collection of literature and detailed review pages.
- **Responsive UI:** Clean, static design that works across desktop and mobile devices.
- **Static Pages:** Includes Home, Browse Books, About Us, Contact, and User Profile pages.

## 🛠️ Tech Stack

This is a static web project built without any heavy frameworks. 

* **HTML5** - Page structure and content
* **CSS3** - Styling, layout, and responsive design
* **JavaScript (Vanilla)** - Basic DOM manipulation and interactivity

## 💻 How to Run Locally

Since this is a static project, there are no dependencies to install or build steps required.
